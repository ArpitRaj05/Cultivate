# **App Name**: Cultivate Focus

## Core Features:

- Habit & Goal Tracker: Users can set and track daily habits and long-term goals with clear progress indicators, emphasizing consistency.
- Interactive Focus Timer: A customizable focus timer with visual feedback to guide users through work sessions, promoting deep concentration.
- Progress Visualization Calendar: A calendar interface to visualize daily habit completion and goal progress, with color-coded marks for success and shared progress views for friends.
- Personalized Focus Assistant: A generative AI tool that provides tailored concentration tips and productivity methods based on user habits and focus patterns.
- Gamified Engagement System: Integrates streak tracking, celebratory animations, and achievement badges to create a highly engaging and 'addictive' loop for completing tasks and maintaining habits.
- Dynamic Theme Switcher: Allows users to toggle between a calming light mode and a focused dark mode, optimizing the visual experience for different preferences and environments.

## Style Guidelines:

- Primary Color: A calming, deep green (#60AC39), symbolizing growth and vitality, chosen to stand out on the light background.
- Background Color: A very light, desaturated green (#EEF5EC), providing a serene and uncluttered canvas for content.
- Accent Color: A vibrant yellow-green (#84E92A), used for key interactive elements and positive feedback to draw attention without being overwhelming.
- Body and headline font: 'PT Sans' (humanist sans-serif) for clear readability and a warm, approachable feel, suitable for extended text and navigation.
- Use clean, minimalist line icons that subtly convey actions and progress without distracting the user, aligning with the app's focus-driven design.
- Maintain a highly minimalist and uncluttered layout with generous white space to minimize distractions and enhance focus on primary content areas like the timer and progress visuals.
- Incorporate subtle, positive micro-animations for task completion, streak milestones, and timer transitions to provide instant, gratifying feedback and reinforce engagement.
